# matrix_7x15 本地构建说明

## 环境要求

- **west**（Zephyr 的元工具）
- **Zephyr SDK**（含 arm-none-eabi 等，并设置好 `ZEPHYR_TOOLCHAIN_VARIANT=zephyr`）
- **Python 3.8+**

## 一键构建（推荐）

在 **thinkpad 仓库根目录** 打开 PowerShell，执行：

```powershell
.\build_matrix_7x15.ps1
```

脚本会：

1. 若尚未拉取 zmk / 模块，先执行 `west update`
2. 使用 `zmk/app` 或 `.zmk/zmk/app` 作为源码目录（自动检测）
3. 以 `config` 为 ZMK_CONFIG，以本仓库根目录和 `kb_zmk_ps2_mouse_trackpoint_driver` 为 ZMK_EXTRA_MODULES
4. 输出到 `build/matrix_7x15/zephyr/zmk.uf2`

## 手动构建

在 thinkpad 根目录执行：

```powershell
# 首次或换板子时建议 -p（清空 build）
west build -s zmk/app -d build/matrix_7x15 -p -b nice_nano -- `
  -DZMK_CONFIG="$PWD/config" `
  -DSHIELD=matrix_7x15 `
  -DZMK_EXTRA_MODULES="$PWD;$PWD/kb_zmk_ps2_mouse_trackpoint_driver"
```

若 ZMK 在 `.zmk/zmk` 下，把 `-s zmk/app` 改成 `-s .zmk/zmk/app`。

## 烧录

构建完成后，将 `build/matrix_7x15/zephyr/zmk.uf2` 复制到键盘进入的 UF2 盘符根目录即可。
